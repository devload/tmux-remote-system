# 🎮 게임 기획서 (Game Design Document)

## 📌 개요

### 게임명 (가칭)
**"Morning Puzzle Battle"** / **"아침 두뇌 배틀"** / **"Puzzle Rivals"**
> 최종 이름은 추후 결정

### 한 줄 컨셉
**"매일 아침 15분, 친구와 3판 퍼즐 대결"**

### 타겟 유저
- 20-40대 직장인
- 출퇴근 시간 활용
- 친구와 가벼운 경쟁을 즐기는 캐주얼 게이머

### 플랫폼
- iOS / Android

### 수익 모델
- 인앱결제 (보석 구매 → 아이템/코스튬)

## 🧩 게임 목록

### 현재 확정 (프로토타입 완료)
| 게임 | 설명 | 예상 플레이 시간 |
|------|------|-----------------|
| Sudoku | 6x6 스도쿠 | 3-5분 |
| Streams | 1-20 숫자 연결 | 3-5분 |
| Hitori | 중복 숫자 제거 | 3-5분 |
| Nurikabe | 섬과 바다 구분 | 4-6분 |

### 추후 추가 예정
- Kakuro (숫자 크로스워드)
- Slitherlink (루프 만들기)
- Nonogram (픽셀 퍼즐)
- Light Up (전구 배치)
- 기타 계속 추가...

### 게임 선정 규칙
- 매일 오전 00:00(자정) 3게임 랜덤 선정
- 같은 그룹 = 같은 3게임
- 난이도도 랜덤 (Easy/Medium/Hard)

## ⏱️ 시간 제한

| 난이도 | 제한 시간 | 특징 |
|--------|----------|------|
| Easy | 5분 | 초보자/워밍업 |
| Medium | 4분 | 기본 난이도 |

- 해당 게임 0점 처리
- 또는 남은 칸 수에 따라 부분 점수?

## 👥 소셜 시스템

### 그룹 (Group)
```
┌─────────────────────────────┐
│  🏠 그룹: "회사 동료들"        │
├─────────────────────────────┤
│  👤 나 (그룹장)              │
│  👤 철수                     │
│  👤 영희                     │
│  👤 민수                     │
│  👤 지연                     │
│  ➕ 친구 초대 (5/10명)        │
└─────────────────────────────┘
```

- **그룹 최대 인원**: 10명 (조절 가능)
- **그룹 생성**: 누구나 가능
- **초대 방식**: 초대 코드 / 링크 공유
- **가입 조건**: 초대받은 사람만 (모르는 사람 X)
- **복수 그룹**: 여러 그룹 가입 가능? (검토 필요)

### 실시간 고스트
게임 중 친구들의 진행 상황이 실시간으로 보임
```
┌─────────────────────────────┐
│  [게임 진행 중]              │
│                             │

│  │ 1 │ 2 │ 3 │  ← 내 퍼즐   │
│  ├───┼───┼───┤             │
│  │   │ 5 │   │             │
│  └───┴───┴───┘             │
│                             │
│  ─────────────────────────  │
│  👤 철수: ████████░░ 80%    │
│  👤 영희: ██████░░░░ 60%    │
│  👤 민수: ████░░░░░░ 40%    │
│  ─────────────────────────  │
└─────────────────────────────┘
```

## 🎁 아이템 시스템

### 아이템 분류

#### 1. 힌트 아이템 (나에게 유리)
| 아이템 | 등급 | 효과 | 보석 |
|--------|------|------|------|
| 작은 힌트 | ⭐ | 빈칸 1개 정답 표시 | 10 |
| 큰 힌트 | ⭐⭐ | 빈칸 3개 정답 표시 | 25 |
| 시간 추가 | ⭐ | +30초 추가 | 15 |
| 실수 무효 | ⭐⭐ | 실수 1회 무시 | 20 |

#### 2. 방해 아이템 (상대방 방해)
| 아이템 | 등급 | 효과 | 보석 |
|--------|------|------|------|
| 안개 | ⭐ | 상대 화면 일부 가림 (3초) | 15 |
| 시간 도둑 | ⭐⭐ | 상대 시간 -10초 | 30 |
| 숫자 뒤집기 | ⭐⭐ | 상대 숫자 일부 숨김 (5초) | 35 |
| 지진 | ⭐⭐⭐ | 상대 화면 흔들림 (5초) | 50 |
| 정전 | ⭐⭐⭐ | 상대 화면 블랙아웃 (3초) | 60 |
| 숫자 셔플 | ⭐⭐⭐⭐ | 입력한 숫자 랜덤 섞기 | 100 |

> 방해 아이템은 등급이 높을수록 강력하고 짜증남 😈

#### 3. 코스튬 (꾸미기)
| 종류 | 예시 |
|------|------|
| 프로필 프레임 | 골드, 다이아몬드, 불꽃 등 |
| 캐릭터 스킨 | 고양이, 강아지, 로봇 등 |
| 이모티콘 | 게임 중 사용 가능 |
| 승리 이펙트 | 1등 할 때 효과 |

### 아이템 사용 규칙
- 게임당 아이템 사용 제한? (예: 최대 2개)
- 방해 아이템은 쿨다운?
- 같은 아이템 중복 사용 불가?

## 🏆 랭킹 & 보상 시스템

### 일일 랭킹
```
┌─────────────────────────────────────┐
│  🏆 오늘의 결과 - "회사 동료들"       │
├─────────────────────────────────────┤
│  🥇 1위  영희    2,450점   +50💎    │
│  🥈 2위  나      2,380점   +30💎    │
│  🥉 3위  철수    2,100점   +20💎    │
│     4위  민수    1,890점   +10💎    │
│     5위  지연    1,650점   +5💎     │
└─────────────────────────────────────┘
```

### 점수 계산
```
게임별 점수 = 기본점수 - 시간패널티 - 실수패널티 + 난이도보너스

총점 = 게임1 + 게임2 + 게임3
```

### 보상 체계

#### 일일 랭킹 보상
| 순위 | 보석 | 추가 보상 |
|------|------|----------|
| 1위 | 50 | 랜덤 아이템 박스 |
| 2위 | 30 | - |
| 3위 | 20 | - |
| 4위~ | 10 | - |
| 참가상 | 5 | - |

#### 주간 랭킹 보상 (선택적)
| 순위 | 보석 | 추가 보상 |
|------|------|----------|
| 1위 | 300 | 희귀 코스튬 |
| 2-3위 | 150 | 아이템 박스 |
| 4-10위 | 50 | - |

#### 연속 출석 보상
| 일수 | 보상 |
|------|------|
| 3일 | 20💎 |
| 7일 | 50💎 + 아이템 |
| 14일 | 100💎 + 희귀 아이템 |
| 30일 | 300💎 + 한정 코스튬 |

## 🛠️ 기술 스택

### 클라이언트
- **엔진**: Unity 2022 LTS
- **언어**: C#
- **UI**: Unity UI Toolkit 또는 UGUI

### 서버
- **기존**: Spring Boot (Java) - 재활용 가능
- **실시간**: WebSocket (고스트 기능)
- **DB**: PostgreSQL / MySQL
- **캐시**: Redis (실시간 데이터)

### 인프라
- AWS 또는 GCP
- 인앱결제: Unity IAP (iOS/Android 통합)
